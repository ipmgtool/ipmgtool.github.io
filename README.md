# IPMG Website

Small GitHub Pages website for IPMG, an IP management and ping monitoring CLI tool.

## Edit

- Main page: `docs/index.md`
- Custom CSS: `docs/stylesheets/extra.css`
- Favicon: `docs/assets/images/favicon.svg`
- Site config: `zensical.toml`

## Build

```bash
.venv/bin/zensical build
```

The generated static site is written to `site/`.

## Preview

Open `site/index.html` in a browser after building.
